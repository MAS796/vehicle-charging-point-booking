‚ö° About EV Charging System

What is this?

A modern full-stack web application for finding and booking EV charging stations with real-time availability, secure booking, and integrated payments.

Technology Stack

Backend: FastAPI (Python), SQLAlchemy ORM, SQLite

Frontend: HTML5, Bootstrap 5, Tailwind CSS, JavaScript

Server: Uvicorn ASGI

‚ú® Key Features
  • Real-time station availability tracking
  • Station opening/closing hours with automatic validation
  • "Book Your Slot" button appears ONLY when station is open
  • Triple-layer validation for security
  • Multiple payment methods (UPI, Card, Wallet)
  • Charger availability management
  • Admin dashboard
  • 30-second live status updates
  • Server-side time validation
  • Responsive design
